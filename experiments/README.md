# Experiments (Research Sandbox)

This directory contains exploratory experiments and artifact traces.
Contents are not part of the core library and may contain large or
generated files.

| Directory | Description |
|-----------|-------------|
| `iree_artifacts/` | IREE compilation artifact traces (pass history, IR dumps) |
| `debug_info/` | Debug information experiments |
| `lit_command/` | LIT test command experiments |
| `torchao/` | TorchAO quantization experiments |
| `VLA/` | Vision-Language-Action model experiments |
